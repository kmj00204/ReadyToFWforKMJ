# 🛡️ 프레임워크 개인 학습 (ReadyToForgeWeapon)

> **나만의 무기 갖기 프로젝트**를 대비하여, 각자 원하는 프레임워크를 학습하는 시간을 갖습니다.  
> 본인에게 적합한 형태의 학습자료로 빠르게 개요를 학습하고, **"개인별 과제"** 혹은 **"팀별 과제"**를 진행합니다.

---

## 📚 권장 프레임워크 (Tech Stack)

* **Frontend**: React
* **Backend**: Node.js, NestJS, Spring Boot (Java/Kotlin)

---

## 💻 과제: 게시판 웹페이지 구현

### Option 1. 개인별 과제
자신이 선택한 프레임워크로 게시판의 **프론트엔드/백엔드**를 구현해봅니다.  
*(클라우드에 배포할 필요는 없습니다.)*

### ⚙️ 기능 요구사항
- [ ] **회원가입 / 로그인** (인증)
- [ ] **게시물 작성**
- [ ] **게시물 목록 보기**
- [ ] **게시물 읽기**
- [ ] **댓글 작성**
- [ ] **페이징** (Pagination)

### 🛠 개발 범위
* **Frontend**: 게시판 UI 구현
* **Backend**: 게시판 Server API 구현
* **Challenge**: *추가적 구현 가능 (기술적 챌린지 요소)*

---

## 👥 인원 구성 및 개발 방식

**혼자 Fullstack 개발**
* 상용 사이트 하나 잡고 프론트엔드와 백엔드를 모두 직접 구현
